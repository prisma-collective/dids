Initial experimental release of Prisma DIDs packages to npm under the `@experimental` dist-tag.

- `@prisma-events/dids-types` — shared TypeScript types and constants
- `@prisma-events/dids-schemas` — VC credential schemas and registry
- `@prisma-events/dids-sdk` — DID lifecycle, VC issuance, and Cardano anchoring SDK

No stable `latest` release yet. Install with `pnpm add @prisma-events/dids-sdk@experimental`.
