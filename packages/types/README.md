# @prisma-events/dids-types

> **Experimental** — Pre-release. Install via `@prisma-events/dids-sdk@experimental`.

Shared TypeScript types and constants for [Prisma DIDs](https://github.com/prisma/prisma-dids).

This package is a dependency of `@prisma-events/dids-sdk`. Most consumers should install the SDK directly:

```bash
pnpm add @prisma-events/dids-sdk@experimental
```
